# 앱 스크린샷 제작 계획

Google Play 등록 전 최소 4장 정도를 준비합니다.

## 추천 스크린샷

1. 홈 입력 화면
   - 문구: 속마음은 그대로, 표현은 프로답게
   - 화면: 원문 입력창, 상대/톤 선택

2. 변환 결과 화면
   - 문구: 안전한 표현부터 단호한 표현까지
   - 화면: 위험도와 추천 표현 3종

3. 자주 쓰는 문장 화면
   - 문구: 자주 쓰는 업무 문장을 바로 변환
   - 화면: 템플릿 버튼 영역

4. 다크모드 화면
   - 문구: 출근길에도 회의 전에도 빠르게 복사
   - 화면: 다크모드 결과 카드

## 이미지 규격 메모

- 휴대전화 스크린샷: 최소 2장 필요
- 권장: 1080 x 1920 또는 1440 x 2560
- 앱 아이콘: 512 x 512 PNG 필요
- 기능 그래픽: 1024 x 500 JPG 또는 PNG 필요
