# Google Play 데이터 보안 답변 초안

이 문서는 Play Console의 데이터 보안 섹션을 작성할 때 참고하기 위한 초안입니다. 실제 제출 전에는 앱에 붙인 광고 SDK, 분석 SDK, 서버 로그 정책에 맞춰 다시 확인해야 합니다.

## 데이터 수집 여부

앱은 사용자가 입력한 문장을 AI 변환 기능 제공을 위해 서버로 전송할 수 있습니다.

## 수집될 수 있는 데이터 유형

- 앱 활동: 사용자가 입력한 문장, 선택한 상대/톤/형식
- 기기 또는 기타 ID: 서버 요청 처리 과정에서 IP 주소, User-Agent 등 기본 요청 정보가 서버 로그에 남을 수 있음

## 수집 목적

- 앱 기능 제공
- AI 문장 변환 처리
- 오류 진단 및 서비스 안정성 개선
- 남용 방지 및 무료 사용량 제한

## 제3자 공유

입력 문장은 AI 변환 처리를 위해 외부 AI API 제공자에게 전송될 수 있습니다.

## 계정 생성

현재 앱은 별도 회원가입이나 로그인을 요구하지 않습니다.

## 출시 전 재확인 필요

- AdMob 또는 다른 광고 SDK를 붙이면 광고 ID와 광고 데이터 항목을 추가해야 합니다.
- Firebase Analytics를 붙이면 앱 활동/기기 ID 관련 항목을 추가해야 합니다.
- 서버 로그 보관 기간을 정하면 개인정보처리방침에 반영해야 합니다.
